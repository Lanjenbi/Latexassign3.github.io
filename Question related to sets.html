I. VERY SHORT ANSWER TYPE QUESTIONS (1 Mark)

1. Find the domain of the given function \(f(x)= \frac{x^2}{x^2 + 3x + 2}\)

2. Find the range of the given function \( f(x)= \frac{3}{2 - x^2}\)

3. Write the given set in set-builder form: \(A = { 5,25,125,625}\)

4. Write the given set in roster form:\(A = {x \in N, x^2 < 25,}\)

5. Write the following intervals in set-builder form:\([-20,3)\)

II. SHORT ANSWER TYPE QUESTION (2 Mark)

1. Let A, B and C be three sets, then prove that: \( A -(B \cap C)= (A - B)\cup (A-C)\)

2. Let A, B and C be three sets, then prove that: \( A \cap (B-C)=(A \cap B)-(A \cap C)\)

3. Let A, B and C be three sets, then prove that:\( (A \cup B)- C =(A - C) \cup (B - C)\)

4. Represent the following relation by venn diagram.\( (a) A - B \qquad (b)(A \cup B)'\)

5. Find the range of the function \( f(x)=\frac{x}{x^2 + 3x + 2}\)

LONG ANSWER TYPE QUESTION (4 Mark)

1. Let A and B be two sets such that \( n (A)=18,n(A \cup B)=45 \text{ and } n(A \cap B)= 6\). Find \( (i) n(B) \qquad (ii) n(A -B) \qquad (iii) n(B-A)\)

2. In a group of 12000 people, there are 800 people who can speak Hindi and 480 people who can speak spanish. How many can speak Spanish only? How many can speak both Hindi and Spanish?

3. Find the range of \( f(x) = 11 + 3 \cos 2x\)

4. If \( A = {1,2,3,4}, B = {3,4,5,6} \text{ and } C = {1,3,5}\),verify that \( A - (B \cup C) = (A - B) \cap (A -C) \)

5. Using properties, if A,B and C are three sets, then prove that \( A - (B - C) = (A - B) \cup (A \cap C )\)
